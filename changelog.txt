# pytends-async changelog

## 0.2.0

2019-12-03

### Changed

- `GetNewProxy()` replaced with internal method `iterate_proxy()`
- Protocol changed from HTTP/2 to HTTP/1.1. This resolves a KeyError that was occurring with the underlying http2 lib.
- HTTP connections are now properly cleaned up.
